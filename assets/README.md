# Demo Assets

This directory contains demo assets for InclusiveAI presentation and testing.

## Directory Structure

```
assets/
├── videos/          # Sample sign language videos
├── audio/          # Sample speech clips
├── images/          # Sample printed pages for OCR
└── README.md        # This file
```

## Assets Needed

### Videos
- `sign_language_sample.mp4` - Sample sign language video for demo
- `hand_gestures_sample.mp4` - Sample hand gesture video

### Audio
- `speech_sample.mp3` - Sample speech clip for STT testing
- `tts_sample.mp3` - Sample TTS output

### Images
- `sample_text.png` - Sample printed text for OCR testing
- `sample_document.jpg` - Sample document for OCR demo

## Usage

These assets are used for:
1. **Demo presentations** - Show judges the system capabilities
2. **Testing** - Test OCR, STT, and sign language detection
3. **Documentation** - Include in README and pitch materials
